Overview

This module allows you to change list views on the fly and thus offers a quick and easy way to view the desired data. Furthermore the List view can be switched from Readonly mode to Editable mode and the current list can be exported directly into an Excel file.

Export current view

Click on the Export button in a List view to export the current view directly to an Excel file.
